# Data Engineer Bootcamp

This repository documents my hands-on training in Data Engineering.  
It contains practical implementations of core concepts such as data ingestion, transformation, database integration, and pipeline architecture using Python and PostgreSQL.

## Tech Stack
- Python
- PostgreSQL
- Pandas
- SQLAlchemy
- Git
